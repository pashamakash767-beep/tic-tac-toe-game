I created a mobile-responsive Tic Tac Toe game using HTML, CSS, and JavaScript.

HTML was used to structure the game board and interface, including a grid layout for the cells and a section to display game results.

CSS handled the design and responsiveness. I used flexbox/grid and media queries to ensure the game works well on both desktop and mobile devices. I also added a linear gradient background to make the UI visually appealing.

JavaScript implemented the game logic. This included:

Alternating turns between players (X and O).

Checking for winning combinations after each move.

Detecting draw conditions when the board is full.

Displaying results on a separate screen (Win or Draw).

A New Game button to reset the board and start fresh.

Key features:

Fully responsive design for different screen sizes.

Dynamic updates without page reloads.

Clear user experience with result display and restart option.

This project demonstrates my ability to combine frontend development skills with game logic implementation, as well as writing clean, modular, and interactive JavaScript code.
